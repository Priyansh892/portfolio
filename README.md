# Priyansh Agarwal - Portfolio

Personal portfolio website built with React.js, showcasing my work as a Full Stack Engineer with 2+ years of experience building production-grade web applications.

## Live Demo
[priyansh-portfolio.vercel.app](https://priyansh-portfolio.vercel.app)

## Built With
- React 18
- CSS3 with custom properties (no UI framework)
- EmailJS / Formspree for contact form
- IntersectionObserver API for scroll animations
- Google Fonts - Syne, Space Grotesk, JetBrains Mono

## Sections
- **Home** - intro, quick stats
- **About** - background, tech stack
- **Experience** - work history at Frisome, Pepperfry, Impelsys
- **Projects** - RozanaNews, Pulzz, ThirdSpace, FoodyGeek
- **Skills** - frontend, backend, cloud & tooling
- **Contact** - contact form + social links

## Running Locally

```bash
git clone https://github.com/Priyansh892/portfolio
cd portfolio
npm install
npm start
```
